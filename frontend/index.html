Adesão de Cliente
Cadastra um novo cliente no produto de Compras Programadas.

POST /api/clientes/adesao

Saída do Produto
Registra a saída de um cliente do produto. A custódia é mantida.

POST /api/clientes/{clienteId}/saida

Alterar Valor Mensal
Altera o valor de aporte mensal de um cliente.

PUT /api/clientes/{clienteId}/valor-mensal

Carteira do Cliente
Exibe a carteira de investimentos com holdings, P&L e composição.

GET /api/clientes/{clienteId}/carteira

Ticker Qtd P. Médio Cotação Atual Valor Atual P&L (R$) P&L (%) Composição
Rentabilidade do Cliente
Exibe histórico de aportes e evolução da carteira.

GET /api/clientes/{clienteId}/rentabilidade

Ordens do Cliente
Lista todas as ordens executadas para o cliente — compras do motor e operações de rebalanceamento.

GET /api/clientes/{clienteId}/ordens

GET /api/admin/cesta/atual

Conta Master — Custódia
Visualiza a custódia consolidada da conta MASTER do broker.

GET /api/admin/conta-master/custodia

Kafka — Eventos de IR
Lista mensagens do tópico Kafka de eventos de Imposto de Renda.

GET /api/admin/kafka/mensagens

Reset do Banco de Dados
Apaga todos os dados de todas as tabelas. Use apenas em ambiente de desenvolvimento!

DELETE /api/admin/reset-database

⚠️ Esta operação é irreversível. Todos os clientes, cestas, custódias e histórico serão apagados.

Motor de Compra
Dispara manualmente o ciclo de compra programada. Processa todos os clientes ativos.

POST /api/motor/executar-compra

Se não informada, usa a data atual. O motor busca cotações, calcula parcelas e distribui para cada cliente ativo.
